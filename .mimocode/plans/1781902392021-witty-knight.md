# Plan: Fix Shortcuts Overlay Close Button

## Problem
The close button (X) on the shortcuts overlay doesn't work - only Escape key closes it. This is due to an event name mismatch:
- Component emits `closed` event
- Parent listens for `close` event

## Fix

### `/mnt/Other/Projects/Tauri/Translator/src/app/pages/translation/translation.page.html`
- Line 97: Change `(close)` → `(closed)`

```html
<app-shortcuts-overlay
  [shortcuts]="showShortcuts()"
  (closed)="showShortcuts.set(false)"  <!-- Fixed: closed not close -->
/>
```
